# 4. 미션 & 사업화

## 미션 기능 (게이미피케이션)

| 미션 | 방식 | 보상/비고 |
|------|------|-----------|
| 5만원으로 여행하기 | 정해진 예산 안에서 코스를 소화하는 도전 | |
| 주요 관광지 방문 | 각 관광지에서 친구들과 인증샷 업로드(처음 키오스크 인물 모두 포함) → 스탬프 | 관광지별 스탬프 |
| 전체 방문 완료 | 모든 관광지 방문 시 스탬프 | 완주 스탬프 |
| 미션 완료 보상 | 스탬프 수에 따른 포인트 제공 | 포인트 정책 논의 필요 |
| 사주 여행 | 그날 운세에 따른 여행 | |

> 메모: 여정 자체가 보상이라서 인증샷만으로 충분히 만족할 것 같음.

## 사업화

| 모델 | 내용 | 비고 |
|------|------|------|
| 광고 수익 | 관련 식당·업소 추천(제휴) | 코스 내 노출 방식 논의 |

## 웹앱 관점 참고 노트

- 스탬프/포인트는 익명 세션 범위에서 유지하거나, 공유 링크 기반으로 그룹에 귀속시키는 방식 검토.
- 인증샷 기반 미션은 웹앱 카메라 + 이미지 카드 생성과 결합하면 SNS 공유 바이럴로 이어질 수 있음.
