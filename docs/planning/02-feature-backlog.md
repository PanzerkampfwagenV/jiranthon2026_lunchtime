## 웹앱 관점 참고 노트

- **F-01 위치 자동 인식**: 키오스크의 고정 위치 대신 웹앱은 브라우저 Geolocation 또는 사용자 지정 출발지로 대체.
- **F-10/F-11 출력**: 웹앱에서는 즉석 인쇄보다 공유 링크·이미지 카드·QR 저장이 자연스러움.
- **영업시간(F-03/F-15)**: 데이터 확보가 난제. MVP에서는 Google Places(유료) 또는 제한적 범위로 축소 검토 필요.
